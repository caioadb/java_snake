# Java Snake

A simple Snake game implementation using Java 25 and Swing.

This project was created for the **567SM - SOFTWARE DEVELOPMENT METHODS 2025** course at the **University of Trieste**.

## Features
- Snake gameplay
- TDD implementation

## Requirements
- Java 25

## How to Run
```bash
./gradlew run
```

## How to Test
```bash
./gradlew test
```
